About This Project

Body

While working on this project, I gained hands-on experience in building a complete Natural Language Processing (NLP) pipeline. I started with raw text data from the Kaggle Fake Job Posting dataset and learned how to transform it into meaningful features that a machine learning model can understand.

I applied several preprocessing steps such as converting text to lowercase, removing punctuation, handling numbers, stopword removal, tokenization, stemming, and normalization. These steps helped me clean the dataset and prepare it for feature extraction.

For feature engineering, I explored both Bag of Words and TF-IDF, but I found TF-IDF to be more effective for this task. I then built a Logistic Regression model, trained it on balanced data, and evaluated it using metrics like Accuracy, Precision, Recall, F1-Score, and Confusion Matrix.

Finally, I integrated the trained model into a Flask web application, where users can paste a job description and instantly check if it is real or fake. This gave me practical exposure to deploying machine learning models in a real-world interface.

Conclusion

In conclusion, this project allowed me to connect theory with practice. I not only understood the importance of preprocessing and feature extraction but also experienced how models are trained, evaluated, and deployed. The Fake Job Detector demonstrates how NLP can be applied to solve real-world problems, and completing this project has strengthened my confidence in working with text data and machine learning systems.
